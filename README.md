# Portfolio Website

One-page portfolio website with a structure inspired by modern developer portfolios.

## Files

- `index.html` — page structure
- `styles.css` — layout and visual design
- `script.js` — dynamic content rendering + smooth interactions

## Run locally

Because this is a static site, you can open `index.html` directly in your browser, or run a lightweight local server.

### Option 1: Open directly

Double-click `index.html`.

### Option 2: Local server (recommended)

If you have Python installed:

```bash
python -m http.server 5500
```

Then open:

`http://localhost:5500`

## Personalize content

Edit portfolio data in `script.js`:

- `portfolioData.profile`
- `portfolioData.stats`
- `portfolioData.stack`
- `portfolioData.experience`
- `portfolioData.projects`
- `portfolioData.social`

## Notes

- This implementation uses original HTML/CSS/JS and placeholder content.
- Replace placeholders with your real text, links, and project images.
