## Run Locally

You can open `index.html` directly, or run a lightweight local server.

### Recommended (Python)

```bash
python -m http.server 5500
```

Then open: `http://localhost:5500`

## Customization

Update your content in `script.js`:

- `portfolioData.profile`
- `portfolioData.stats`

You can further tune layout and visuals in `styles.css`.
